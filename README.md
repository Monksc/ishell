#Description
If you have to use a shell that sucks like spim or jdb and doesn't come 
with the ability to clear the screen. Or is missing features likea history and
editing the first line.

# Usage

```
gcc main.c
```

then run it
```
./a.out <bad shell> <arguments>
```

an example is
```
./a.out mips
```

```
./a.out jdb ClassFile
```
